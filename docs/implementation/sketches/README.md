# Adapter Implementation Sketches

Task-specific adapter sketches live here when a component issue needs
counterpart exploration before coding.

Create one sketch per adapter task from:

```text
docs/implementation/adapter-components/templates/reference-exploration-sketch.md
```

Use the sketch as the implementation guide and update it through handoff. Do
not place generated browser artifacts here; store snapshots and screenshots
under `.playwright-cli/` or `/tmp/`, then reference those paths from the
sketch.
